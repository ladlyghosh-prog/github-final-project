# Simple Interest Calculator

A lightweight web-based tool to calculate the simple interest on a principal amount over a set period.

## Formula Used
The simple interest is calculated using the formula:
SI = (P * R * T) / 100

- **P (Principal):** The initial amount of money.
- **R (Rate):** The annual interest rate (percentage).
- **T (Time):** The time the money is invested or borrowed for (in years).

## Features
- Fast and accurate calculations.
- Clean and readable documentation.
- Easy-to-understand logic.
